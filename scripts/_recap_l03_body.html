What attributes are

Every object stores its data in named variables called attributes.

02

What methods are

A method is a function inside a class that acts on its own data.

03

Attributes give objects memory

Each object you create holds its own independent copy of every attribute.

04

Methods act on that data

Calling a method on any object reads or updates that specific object's attributes.
