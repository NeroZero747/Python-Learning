Overview

A high-level summary of the topic

Py

Python is a programming language — a way to write instructions that a computer can understand and execute.

What can you do with Python?

Analyze data
Automate tasks
Process files
Query databases
Build dashboards
Connect to APIs
Easy to learn

Python reads like plain English — no semicolons, no curly braces. It's widely considered the most beginner-friendly language.

General purpose

Unlike Excel or SQL, which focus on specific tasks, Python can handle data analysis, web apps, automation, and more — all in one language.

This is why Python is the #1 language in data science, data engineering, and analytics — and why we're starting here.

Decision Flow

A visual walkthrough of the logic and process

Python reads your file top to bottom, executing one instruction at a time. Here's the path every script takes:

Run Script

You hit "Run"

Read Line

Check & execute

More Lines?

Decision point

Next Line

YES → repeat

Done

NO → exit

If YES → loops back to "Read Line" and repeats until no more lines
script.py 
name = "Alice"
greeting = "Hello, " + name
print(greeting)
Terminal $ python script.py
Hello, Alice
1

Store a value

name = "Alice" saves the text into a variable

2

Combine strings

"Hello, " + name joins text together

3

Print the result

print() outputs to the terminal

Order matters — swap line 1 & 2 and you'd get an error

Common Mistakes

Frequent errors and how to avoid them

Forgetting quotation marks

Strings must use quotation marks.

SyntaxError

Incorrect 

print(Hello)

Correct 

print("Hello")

Text values (strings) must always be wrapped in quotes — either "double" or 'single'.

Misspelling Python commands

Python is case-sensitive — commands must be spelled exactly.

NameError

Incorrect 

pritn("Hello")

Correct 

print("Hello")

Python is case-sensitive. print works, but Print or pritn will cause an error.

Missing parentheses

Function calls require parentheses around arguments.

SyntaxError

Incorrect 

print "Hello"

Correct 

print("Hello")

In Python 3, print is a function — always use parentheses: print("text").
