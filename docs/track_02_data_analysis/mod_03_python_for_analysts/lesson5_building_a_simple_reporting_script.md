# Overview

Many analysts create reports by manually performing several steps.

Example workflow:

```text
Download raw dataset
Clean data
Calculate metrics
Create summary tables
Export report
```

This process might take **30–60 minutes each time the report is created**.

Python allows analysts to combine all these steps into a **single automated script**.

Example automated pipeline:

```text
Raw Data
    ↓
Python Script
    ↓
Clean Dataset
    ↓
Calculated Metrics
    ↓
Excel Report
```

Once the script is created, the entire report can be generated by running:

```bash
python report_script.py
```

# Key Concepts

## Step 1 — Extract Data

The first step in a reporting script is retrieving the source data.

Example reading a CSV file:

```python
import pandas as pd

df = pd.read_csv("sales_data.csv")
```

Example output:

| region | sales |
|---|---|
| East | 200 |
| West | 150 |
| South | 300 |

---

## Step 2 — Transform Data

After loading the data, Python can perform calculations and transformations.

Example creating a new column:

```python
df["tax"] = df["sales"] * 0.08
```

Example calculating totals:

```python
total_sales = df["sales"].sum()
```

---

## Step 3 — Generate Summary Tables

Python can aggregate data similar to pivot tables.

Example:

```python
summary = df.groupby("region")["sales"].sum()

print(summary)
```

Output example:

| region | sales |
|---|---|
| East | 200 |
| West | 150 |
| South | 300 |

---

## Step 4 — Export the Report

The final step is exporting the results.

Example exporting to Excel:

```python
summary.to_excel("sales_report.xlsx")
```

The script now generates the final report automatically.

# Common Mistakes

### Skipping Data Validation

Always check data before generating reports.

Example:

```python
print(df.head())
```

---

### Overwriting Original Data

Always write output files separately.

Example:

```text
sales_data.csv → input
sales_report.xlsx → output
```

---

### Forgetting to Document Scripts

Scripts should contain comments explaining each step.

Example:

```python
# Calculate sales tax
df["tax"] = df["sales"] * 0.08
```
